Setup Guide
Connect The Ledger to Google Sheets in ~5 minutes.
1
Create a Google Sheet
Go to sheets.new — name it anything, e.g. "The Ledger".
2
Add the Apps Script
In your Sheet: Extensions → Apps Script. Delete all code. Paste the contents of apps-script.js. Change hashPin('1234') to your PIN.
3
Deploy as Web App
Deploy → New deployment
Type: Web app · Execute as: Me · Who can access: Anyone
Copy the Web App URL.
4
Paste URL here
Saved in this browser — only needs to be done once per device.
